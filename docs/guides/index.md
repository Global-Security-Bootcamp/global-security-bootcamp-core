- Speakers and Lab Facilitators

### Branding Your Event
Maintain consistency while adding local flavor:
- Naming convention: "Global Security Bootcamp — [City] [Year]"
- Logo usage and color palette
- Co-branding with sponsors
- Social media and website guidelines

### Building Sponsor Relationships

- 3 sponsorship tiers (Supporter, Partner, Principal)
- Sponsorship agreement template
- Recruitment strategies
- Communication timeline

### Setting Community Standards
Create an inclusive, welcoming environment:
- Expected behaviors and conduct
- Reporting procedures
- Confidentiality and enforcement
- Participant and organizer responsibilities

### Going Global
Reach international audiences:
- Translation workflow
- Language folder structure
- Terminology guidelines
- Contribution process for translators
