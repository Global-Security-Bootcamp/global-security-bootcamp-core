# FAQ

Q: How do I adapt the framework to a local event?
A: Copy relevant templates from `templates/` to your event repo, update branding, and run the `standards-compliance` checks.

Q: Where should materials live?
A: Event-specific materials should be housed in a dedicated event repo; reference the core templates and docs via links.

Q: How are translations managed?
A: See `localization/` and `scaffold/localization-guide.md` for guidance and recommended workflows.
