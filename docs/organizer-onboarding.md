# Organizer Onboarding

Purpose: Orient new organizers to the framework, roles, and readiness checkpoints.

Roles
- Steward: maintains core templates and governance.
- Organizer: operates a local event using the framework.
- Reviewer: validates materials against standards.

Readiness checkpoints
- Confirm event objectives and audience.
- Finalize agenda and lab materials.
- Confirm registration and privacy handling.
- Validate communications and runbook.

Documentation norms
- Keep content short, actionable, and process-driven.
- Use templates from `templates/` and update `docs/` when adding new program themes.
