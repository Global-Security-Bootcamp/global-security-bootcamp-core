# Program Themes

Focused, practical cloud-security topics for hands-on learning. Suggested sessions below can be mixed into tracks or labs.

- Identity & Access: identity lifecycle, privileged access, authentication and federation.
- Endpoint & XDR: endpoint detection, telemetry quality, response workflows.
- Cloud Workload Security: container and VM hardening, runtime protection, threat modeling.
- SIEM / SOAR: log ingestion, alert tuning, playbooks and automation.
- Secure DevOps: secure CI/CD pipelines, SCA, secrets management, infrastructure as code security.
- Data Security & Compliance: data classification, encryption, privacy-by-design, regulatory mapping.
- AI Risk & Governance: model risk management, data lineage, prompt-security considerations.
- Zero Trust: segmentation, least privilege, continuous verification.

Emphasize labs and scenario-based exercises that replicate real-world operations.
