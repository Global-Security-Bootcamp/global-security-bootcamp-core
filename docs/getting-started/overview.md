# Getting Started Overview

## Who is this for?

This framework is designed for:

- **Event organizers** invited to run a Global Security Bootcamp in your city or organization.
- **Stewardship team members** maintaining the core framework.
- **Sponsors and partners** supporting community-driven security education.

## What you'll learn

This section covers:

1. **Quickstart** — A 13-step guide to create your own event repository and launch an event.
2. **Create Your Event Repo** — Detailed steps to set up a GitHub repository for your event with the proper structure.
3. **Planning Your Event** — How to align with program themes and event standards.
4. **Using Templates** — How to adapt agendas, communications, and operations for your event.

## Your first steps

1. Read the [Quickstart](quickstart.md) to understand the overall workflow.
2. Follow [Create Your Event Repo](create-event-repo.md) to set up your repository.
3. Explore the [Templates](../templates/overview.md) and customize them for your city, dates, and topics.
4. Review [Event Standards](../planning/event-standards.md) to ensure quality.
5. Use the [Organizer Onboarding](../guides/organizer-onboarding.md) guide for roles and readiness checkpoints.
