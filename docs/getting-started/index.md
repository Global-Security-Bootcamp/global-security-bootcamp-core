## What is a Global Security Bootcamp?

A hands-on, community-driven cloud security learning event designed to:
- Build practical skills across 8 core security domains
- Connect security professionals with local communities
- Create inclusive, accessible learning experiences
- Maintain consistency while allowing local customization
